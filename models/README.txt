Sequence for generating model simulations

Model simulations for angular and radial modulated gratings:

1. makeExptStim.m to create exptStim_angular_scanRoom.mat and exptStim_radial_scanRoom.mat, which already provided in the folder.

2. create_stimScreen.m to create final_stimScreen_angular_scanRoom.mat and final_stimScreen_radial_scanRoom.mat, which already provided in the folder.

3. generate_modelOutput.m

4. simulate_voxelResp.m

5. spatial_recon_model.m

6. plotFigures_spatialRecon_model.m


Model simulations for lines:
1. generate_modelOutput_lines.m

2. simulate_voxelResp_lines.m

3. spatial_recon_model.m

4. plotFigures_spatialRecon_model.m


Model simulations for dots:
1. generate_modelOutput_dots.m

2. simulate_voxelResp_dots.m

3. spatial_recon_model.m

4. plotFigures_spatialRecon_model.m
